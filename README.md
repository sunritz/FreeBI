# FreeBI

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

## Description
[FreeBI](http://www.i-mprove.cc/freebi)是基于[DataV](http://datav.jiaminghi.com/)设计的一款可以自由选择图表展示数据的小工具，你可以参照DataV的各种图表的config参数自定义图表的外形、颜色、图标、数量、单位等，同时还可以自定义图表的大小，画布的背景色，盒子的背景色以及是否启用盒子圆角来自由设计所需要的数据大屏展示。在每项数据设置的底部还可以自己添加数据服务地址以展示自己的数据，最后保存下来的数据就是一张你自己设计的完整的数据大屏展示了。
